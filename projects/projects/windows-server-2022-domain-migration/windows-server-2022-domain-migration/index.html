Infrastructure Project

Windows Server 2022 Domain Controller Migration

This project involved building a new Windows Server 2022 domain controller and migrating users, laptops, network services, and shared data from an older environment.

The new environment included Active Directory Domain Services, DNS, DHCP, Microsoft Entra ID sync, data migration with Robocopy, user migration support with ADManager Plus, and PowerShell automation deployed through ManageEngine.

Privacy note: All names, domains, IP addresses, paths, screenshots, usernames, and company details are sanitised or recreated in a lab format.

Endpoint Migration

Windows 11 Pro Domain Join Automation

I created a PowerShell script to join Windows 11 Pro laptops to the new domain. The script was uploaded into ManageEngine so it could be run more easily across devices instead of manually joining each laptop one by one. 

$DomainName = "example.local"
$OUPath = "OU=Laptops,DC=example,DC=local"

Add-Computer -DomainName $DomainName -OUPath $OUPath -Restart
  • Created a PowerShell domain-join script.
  • Uploaded the script into ManageEngine for easier execution.
  • Joined Windows 11 Pro laptops to the new domain.
  • Confirmed laptops appeared in Active Directory.
  • Checked user sign-in after domain join.
  • Verified DNS and DHCP settings after migration.
  • Troubleshot failed joins where needed.
